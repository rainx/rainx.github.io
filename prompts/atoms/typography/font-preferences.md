- sans-serif 字体优先
- 中文：Source Han Sans, Noto Sans Simplified Chinese(alternative), PingFang SC(Apple System only)
- 亚洲其它： Noto Sans 系列
- 英文：Inter, Open Sans, SF Pro(Apple System only)
- Monospaced：Fira Code
- Web UI 开发时，需要中文时可以使用类似 Apple 的 "SF Pro SC", "SF Pro Text", "SF Pro Icons", "PingFang SC", "Helvetica Neue", "Helvetica", "Arial", sans-serif; ， 如果需要 Webfont 英文可以使用 Google Font载入 Inter
