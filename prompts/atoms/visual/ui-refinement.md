---
id: ui-refinement
name: UI 精细化实践
category: visual
tags: [ui, ux, design, web, refinement]
description: Web UI/UX 视觉精细化最佳实践
---

- 按钮、卡片的边框避免实色 border，改用半透明 ring（如 `ring-1 ring-black/10`），更轻盈通透
- 标题字重避免最粗档（700-900），使用 500-600 并收紧字间距（`tracking-tight`），更精致克制
- 使用 Inter 字体时，大标题开启 `font-variation-settings: "opsz" 32` Display 模式，优化大字号渲染
- 分区小标签使用等宽字体，搭配全大写（可选）+ 宽字间距（`tracking-widest`），增强标签辨识度
